```toml
[dependencies]
modbus = "0.1"
```

## Building modbus-rs

First, you'll need to install _clang_. Afterwards, just run:

```sh
$ git clone https://github.com/zzeroo/modbus-rs
$ cd modbus-rs
$ cargo build
```
